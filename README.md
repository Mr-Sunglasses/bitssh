Terminal user interface for SSH. It uses ~/.ssh/config to list and connect to hosts.

## Installation

Install my-project with pip

```bash
  pip3 install bitssh
  
  bitssh
```

Install from source

```bash
  git clone https://github.com/Mr-Sunglasses/bitssh

  cd bitssh

  python3 setup.py install

  bitssh
```
